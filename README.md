#  Screen Printed Electronics — SPE & IDE Electrode Fabrication

A research project focused on the design and fabrication of **Screen Printed Electrodes (SPE)** and **Interdigitated Electrodes (IDE)** for electrochemical sensing applications, conducted at the **SRRC Lab, Manipal Institute of Technology**.

---

##  Overview

Screen printed electronics offer a low-cost, scalable method for fabricating electrochemical sensors. This project involved setting up and calibrating a screen printing process to produce SPEs and IDEs, with test prints on paper and final prints on Kapton/PET substrate, targeting biosensing and electrochemical detection applications.

---

##  Objectives

- Design and develop a screen printing setup for fabricating SPEs and IDEs
- Optimize process parameters for consistent, high-quality prints
- Establish a repeatable fabrication workflow for electrochemical electrodes
- Validate print quality on paper before printing on final substrate

---

##  Process Details

| Parameter | Details |
|-----------|---------|
| Electrode Types | Screen Printed Electrodes (SPE), Interdigitated Electrodes (IDE) |
| Mesh Size | 350 mesh |
| Exposure Method | UV exposure for screen development |
| Substrate | Kapton / PET (Polyethylene Terephthalate) |
| Test Prints | Paper substrate (for process validation) |
| Lab | SRRC Lab, Manipal Institute of Technology |
| Application | Electrochemical / Biosensing |

---

##  Work Done

- Designed and developed the screen printing setup for SPE and IDE fabrication
- Selected and calibrated 350 mesh screens for fine electrode pattern resolution
- Optimized process parameters including squeegee pressure, mesh alignment, and UV exposure time
- Performed UV exposure for screen development and pattern transfer
- Validated print quality on paper substrates before final substrate printing
- Successfully printed electrode patterns on Paper and Kapton/PET substrate
- Conducted testing and troubleshooting to reduce print defects and improve repeatability
